### 🏃 **꾸준함을 바탕으로!** 🏃
- 2년 동안 주 2회, 총 210건의 기술 정보 포스팅을 진행하였습니다.
- 또한, 현장 실습에서의 꾸준함을 인정받아 봄학기를 넘어 여름학기까지 연장하였습니다.
- 이러한 **꾸준함을 바탕으로**, 카카오 테크 캠퍼스에서 4월부터 8월까지 지속될 긴 호흡을 해내겠습니다.
---
### 🙋‍♂ **적극적인 소통!** 🙋‍♂
- 현장 실습 중 “[고성군의회 웹사이트 개편 프로젝트](https://council.gwgs.go.kr/)”에 중간 투입됨.
- 정보 공유의 제한으로 프로젝트 진행 상황 파악과 해결책 모색에 어려움 겪음.
- 프로젝트 매니저와 **적극적인 소통**을 통해 문제 해결을 시도함.
- [유사한 웹사이트 벤치마킹](http://www.ygcl.go.kr/portal/index)을 통해 아이디어 도출에 동의함.
- 제안된 변화를 통해 웹사이트 사용자로부터 긍정적인 피드백을 받고 게시판 활용도 70% 증가함.
- 문제 해결을 위한 적극적인 소통의 중요성을 깨닫고, 카카오 테크 캠퍼스에서도 협업 정신을 실천하기로 함.
### 카카오 테크 캠퍼스 Q&A 오픈 채팅의 질문에 적극적으로 답변하며 협업 정신을 실천 하고 있음.
